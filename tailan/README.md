1. `LaTeX Workshop` extension suulgana.
2. [main.tex](/home/temuka/code/diplom/tailan/main.tex)-ee neene.
3. `xelatex` ashigladag build recipe songood `Ctrl+Alt+B` darj compile hiine.
4. Preview-g `Ctrl+Alt+V` ashiglaj harna.

Hervee terminalaas compile hiih bol:

```bash
cd /home/temuka/code/diplom/tailan
xelatex main.tex
```

Mongol texttei ajillah bol `xelatex` ashiglakh ni zuv.
